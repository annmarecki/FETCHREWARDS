Hello, the following routes are created!

GET route for all users and their points: http://localhost:3003/api/users

GET route for a single user and their points: http://localhost:3003/api/users/:userId

GET route for all transactions: http://localhost:3003/api/transactions

GET route for a single transaction: http://localhost:3003/api/transactions/:transactionId

POST route for a single transaction with the type of pay: http://localhost:3003/api/transactions/pay

POST route for a single transaction with the type of spend: http://localhost:3003/api/transactions/spend
